正式環境

基本上，FT2018 企業內部自主開發主要以 Web App 發佈，佈署在企業內部的虛擬機，除非有特別說明，一般僅限局網訪問。

用戶使用【電腦|平板|手機】的流覽器，連上局網，必要的登入信息，即可訪問。

開發環境

開發人員的電腦基本上都具有模擬正式環境的功能。 開發環境可以是搭建在 Windows 或是 Linux 的作業環境。

目前的程式開發語言和框架以 Django/Python 為主，也包括 PHP 和 Java。 開發環境可以是搭建在 Windows 或是 Linux 的作業環境。

數據庫

項目相關直接連接的數據庫包括 Oracle, PostgreSQL 和最簡易的SQLite。

需求說明

經過和項目需求人員的訪談，我們會簡述需求供各層管理人員的參考，以利在不同階段有不同的指示調整應變。

技術要點

在現有成熟的開發技術上繼續延伸的可控性是最高的，不存在技術瓶頸。

當現有開發技術或平台組合不能合直滿足需求時，就要進一步嘗試新的技術和平台。在合理的時限內，先評估可行性及必要的開發部署成本。

這裡講的必要的開發部署成本，通常就是人力的投入，必要的自學或外部培訓，內部可應用的資源，或是需要增購或租用。

開發要點

主要先實現可充份實現的部份。

不容易實現的部份將儘量先做出基礎版本，再循序討論優化。
